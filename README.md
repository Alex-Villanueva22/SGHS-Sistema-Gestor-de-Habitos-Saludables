# 🥗 GestHabit: Sistema Gestor de Hábitos Saludables

> Una aplicación web orientada al bienestar estudiantil que combate el estrés académico mediante la gamificación y la gestión inteligente de hábitos saludables.

---

## 🎯 El Desafío
Los estudiantes universitarios suelen descuidar su salud debido a la carga académica. 
* **Problema:** Altos niveles de estrés que derivan en mala alimentación, falta de sueño y sedentarismo.
* **Solución:** Una plataforma interactiva que motiva al usuario mediante un sistema de logros, recordatorios personalizados y seguimiento de progreso diario.

## 🛠️ Stack Tecnológico
* **Frontend:** React.js + Vite (para una experiencia de usuario rápida y fluida).
* **Backend & DB:** Firebase (Authentication, Firestore para datos en tiempo real y Hosting).
* **Lenguaje:** JavaScript (ES6+).

## 🧠 Metodologías de Desarrollo
Este proyecto destaca por su enfoque profesional en el ciclo de vida del software:
* **Design Thinking:** Aplicado en la fase de prototipado para empatizar con el estudiante y definir soluciones reales.
* **SCRUM:** Gestión del desarrollo mediante Sprints, permitiendo entregas incrementales de valor y una organización eficiente del equipo.

## 🧪 UX & Usabilidad (Data-Driven Design)
A diferencia de un proyecto común, este sistema fue validado rigurosamente mediante pruebas de usuario:
* **Pruebas de Usabilidad:** Uso de herramientas como **Maze, Microsoft Clarity, Useberry y Mouseflow** para analizar mapas de calor y flujos de usuario.
* **Accesibilidad:** Optimización del sitio bajo los estándares de **Google Lighthouse**, garantizando una experiencia inclusiva.
* **Analítica:** Implementación de métricas con **Contentsquare y UXTweak** para iterar sobre el diseño original.

## ✨ Características Principales
* ✅ **Sistema de Logros:** Gamificación para incentivar la constancia en los hábitos.
* ✅ **Gestión de Tareas Personalizadas:** Creación de actividades con prioridad (Baja, Media, Alta) y horarios específicos.
* ✅ **Registro de Progreso:** Visualización clara de la evolución del usuario día tras día.
* ✅ **Sincronización en Tiempo Real:** Gracias a la integración con Firebase.
